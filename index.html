What is Java Sparrow

A tiny passerine bird that is also referred to as the Java finch, Java rice sparrow, or Java rice bird.

Where does it habitat?

Java sparrow is a resident breeding bird in Java, Bali and Bawean in Indonesia.

How big is it?

Java sparrow is about 15 to 17 cm in length from the beak to its tip of tail feathers.

What does it eat?

Java sparrow feeds mainly on grain and other seeds.

Can it speak?

No, they can't. But sounds of their chirping voice are beautiful.

People have loved java sparrow as a cage bird for a long time.
